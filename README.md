# Погода

Погодное приложение для головного устройства автомобиля (Android, Каспий/Актау).
Места настраиваются на онбординге и в разделе «Места» (по умолчанию — дом и море),
их можно добавлять сколько угодно; прогноз на 2 недели, карта ветров и осадков,
температура воды. Светлая тема под белый салон, работает без Google-сервисов,
оффлайн показывает последние данные.

## Как пользоваться

- Сверху **места** (чипсы) — переключают локацию; настраиваются на онбординге и
  в разделе «Места» слева.
- **Список по дням** — на неделю(и) вперёд: погода, температура, ветер, вода.
  Тапни по дню → **детали**: график (переключается: температура / ветер / вода /
  влажность) и таблица по часам.
- Слева **меню**:
  - **Погода** — список дней.
  - **Карта** — ветер и осадки на карте, сверху переключаются слои.
  - **Вода** — реальная температура воды с датчиков (lada.kz).
  - **Фильтры** — выбрать период (календарь), шаг (1 ч / 3 ч) и какие колонки
    показывать. Нажал «Применить» — сохранится.
- Нет интернета — покажет последний прогноз со штампом времени, не пустой экран.
- Обновляется само; кнопка **⟳** справа сверху — обновить вручную.

## Установка на голову

1. Открой в браузере головы страницу релизов репозитория → **Releases**.
2. Скачай `app-debug.apk`, открой, разреши установку из неизвестных источников.
3. Готово — иконка «Погода».

Тот же путь, что использовался для SmartTube.

## Разработка

    git clone https://github.com/bykhoda/weather-for-father.git
    cd weather-for-father
    ./gradlew assembleDebug     # или Run в Android Studio

Эмулятор — образ **AOSP** (без Google Play), landscape.

### Свои координаты и ключи — `local.properties` (в git не попадает)

    DACHA_LAT=43.4393       # Ивушка
    DACHA_LON=51.3192
    ROUTE_BEARING=148.4
    OWM_KEY=                 # необязательно, для растровой карты OpenWeatherMap

Без этого файла обе карточки показывают Актау — приложение всё равно собирается.

### Релиз

Пуш тега `v*` собирает APK в **Releases** (GitHub Actions):

    git tag v0.1.2
    git push origin v0.1.2

## Данные

- [Open-Meteo](https://open-meteo.com) — погода, ветер, волны (без ключа).
- [lada.kz](https://www.lada.kz/temperatura_vody_v_more.html) — температура воды с датчиков.
- [Windy](https://windy.com) — карта ветров и осадков (встроена через WebView).
