# Перед сдачей
1. ./gradlew assembleDebug testDebugUnitTest — зелено
2. ./gradlew :app:dependencies | grep gms — пусто
3. Запуск на AOSP-образе эмулятора (БЕЗ Google APIs) — не падает
4. Отключить сеть на эмуляторе -> показан кэш, не спиннер и не диалог
5. Очистить данные + отключить сеть -> внятное пустое состояние
6. Проверить, что цифра 9 -> 11 не сдвигает layout (tnum)
7. Скриншот в градациях серого: три статуса различимы
