# Головное устройство Denza N9
Не Android Automotive. Кастомный Android на планшете в панели.

## Запрещённые зависимости
Любой `com.google.android.gms.*`, Firebase, Play Integrity, FusedLocationProvider.
Проверка: `./gradlew :app:dependencies | grep gms` — должно быть пусто.

## Разрешения
Только INTERNET и ACCESS_NETWORK_STATE (обе normal, диалога не вызывают).
Ни одного runtime-permission. Координаты захардкожены через BuildConfig.
Диалог разрешений на экране машины — провал UX.

## Экран
Только landscape. Тач-таргеты >= 64dp. Основные цифры >= 72sp.
Прямое солнце: контраст >= 7:1, начертание >= Bold.

## Сеть
OkHttp, таймауты 10s/10s. Кэш рисуется ДО сетевого запроса, всегда.
Ошибка сети -> штамп свежести меняет цвет. Не диалог.

## Отладка
adb logcat может быть недоступен. Вся диагностика — на скрытом debug-экране
(долгий тап по заголовку «ЕХАТЬ?»).
